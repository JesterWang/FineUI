###关于FineUI
------------
基于 ExtJS 的专业 ASP.NET 控件库。

###FineUI的使命
------------
创建 No JavaScript，No CSS，No UpdatePanel，No ViewState，No WebServices 的网站应用程序。

###支持的浏览器
------------
Chrome、Firefox、Safari、IE 8.0+
